DataCite Kernel 4.6

This site hosts the JSON-LD contexts and JSON Schema files for the DataCite Kernel 4.6 metadata model.

Controlled vocabularies

  • contributorTypes.jsonld (JSON‑LD vocabulary)
  • dateTypes.jsonld (JSON‑LD vocabulary)
  • descriptionTypes.jsonld (JSON‑LD vocabulary)
  • nameTypes.jsonld (JSON‑LD vocabulary)
  • funderIdTypes.jsonld (JSON‑LD vocabulary)
  • relatedIdTypes.jsonld (JSON‑LD vocabulary)
  • relationTypes.jsonld (JSON‑LD vocabulary)
  • resourceTypeGeneral.jsonld (JSON‑LD vocabulary)
  • titleTypes.jsonld (JSON‑LD vocabulary)

Main context and schemas

  • main_context.jsonld (JSON‑LD context)
  • rules.json (JSON Schema, minimal validator)
  • datacite4.6-schema.json (JSON Schema, full validator)
  • datacite4.6-profile.json (Bundled profile with rules, vocabularies and context)

SKOS Mappings & Crosswalks

All SKOS-based crosswalks live in the skos-mappings/ folder:

  • SKOS_crosswalks.jsonld (SKOS crosswalks)
  • jskos-mappings.json Crosswalks in JSKOS format

You can find the source repository on GitHub.
